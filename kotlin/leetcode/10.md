## 解題思路

這一題考驗的是實作 Regex

雖然這題難度被標記為 Hard

不過利用 Kotlin 已有的 Regex 實作

可以很簡單的回答出這一題

## Kotlin 參考解答

<details>
  <summary>點擊展開解答</summary>


單一表達式的參考做法如下

```kotlin
class Solution {
    fun isMatch(s: String, p: String) =
        p.toRegex().matches(s)
}
```
