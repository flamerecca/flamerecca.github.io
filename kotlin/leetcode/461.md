## 解題思路

這題考驗的是對 binary operation 的熟悉程度

我們先透過 xor 找出距離

然後透過一些運算

來將距離從  1 bit 個數轉換成數字

```kotlin
class Solution {
    fun hammingDistance(x: Int, y: Int): Int {
        var foo = x xor y
        var dist = 0
        while (foo != 0) {
            dist++
            foo = foo and (foo - 1)
        }
        return dist
    }
}
```
