## 解題思路

這一題考的是對字串的處理

藉由將字串當作陣列看待

我們可以很簡單的處理這個問題

## Kotlin 參考解答

單一表達式的方式如下

```kotlin
class Solution {
    fun isAnagram(s: String, t: String) =
        s.toCharArray().sorted() == t.toCharArray().sorted()
}
```
