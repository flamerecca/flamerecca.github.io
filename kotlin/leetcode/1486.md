## 解題思路

這一題考的是陣列內 `xor` 處理

單一表達式的解法如下

```kotlin
class Solution {
    fun xorOperation(n: Int, start: Int) =
        List(n) { start + 2 * it }
            .foldRight(0) { acc, c -> acc xor c }
}
```
