class Solution {

    }

## 解題思路

利用陣列已經排序好這件事情

我們可以只比對原本陣列前後的元素

撰寫時要注意最尾端的元素如何調整

```kotlin
class Solution {
    fun removeDuplicates(nums: IntArray): Int {
        if (nums.isEmpty() || nums.size == 1) {
            return nums.size
        }
        var j = 0
        for (i in 0 until nums.size - 1) {
            if (nums[i] != nums[i + 1]) {
                nums[j] = nums[i]
                j++
            }
        }    
        nums[j] = nums[nums.size - 1]
        j++
        return j
    }
}
```
