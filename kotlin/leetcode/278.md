## 解題思路

這一題考的是二分法的思路

雖然題目說的很複雜

但是只要我們使用二分法

就可以快速找到壞掉的最初版本

## Kotlin 參考解答

```kotlin
class Solution: VersionControl() {
    override fun firstBadVersion(n: Int) : Int {
        var high = n
        var low = 0
        
        while (low < high) {
            val mid = low + ((high - low) / 2)
            if (isBadVersion(mid)) {
                high = mid
            } else {
                low = mid + 1
            }
        }
        return high
    }
}
```
