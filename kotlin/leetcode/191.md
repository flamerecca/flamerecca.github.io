class Solution {

    }

## 解題思路

這題和 [461. Hamming Distance](461.md) 一樣

考驗的是對 binary operation 的熟悉程度

-----

我們可以發現數字 n 和數字 n-1

剛好會在結尾的所有 0 變成 1

並將最接近尾端的 1 變成 0

所以 n and (n-1) 

就會將最接近尾端的 1 變成 0

-----

我們只要計算進行幾次操作之後 n 變成了 0

就可以知道 n 原本包含了幾個 1 bits

```kotlin
class Solution {
    fun hammingWeight(n:Int):Int {
        var count = 0
        var input = n
        while(input != 0) {
            input = input and (input - 1)
            count++
        }
        return count
    }
}
```
