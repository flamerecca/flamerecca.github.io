## 解題思路

這一題考的是對字串陣列的處理

我們可以利用 `map` 的概念來處理所有的單字

放到 Kotlin `Set` 裡面來移除重複的單字

## Kotlin 參考解答

```kotlin
class Solution {
    fun uniqueMorseRepresentations(words: Array<String>): Int {
        val morseDictionary = arrayOf(".-", "-...", "-.-.", "-..", ".", "..-.", "--.", "....", "..", ".---", "-.-", ".-..", "--", "-.", "---", ".--.", "--.-", ".-.", "...", "-", "..-", "...-", ".--", "-..-", "-.--", "--..")

        return words.map { it.map { c -> morseDictionary[c - 'a'] }.joinToString("") }
                .toSet().size
    }
}
```
