## 解題思路
這題題目是字串基礎題

可以很簡單的用 `for` 或 `forEach` 解答

## Kotlin 參考解答
使用 `for` 的方式

```kotlin
class Solution {  
    fun checkIfPangram(sentence: String): Boolean {  
        val lettersMap = mutableMapOf<Char, Char>()  
        for(element in sentence) {  
            lettersMap[element]  = ' '  
        }  
        return lettersMap.keys.size == 26  
    }  
}
```

使用 `forEach` 的方式

```kotlin
class Solution {
    fun checkIfPangram(sentence: String): Boolean {
        val lettersMap = mutableMapOf<Char, Char>()
        sentence.forEach { lettersMap[it] = ' ' }
        return lettersMap.keys.size == 26
    }
}
```
