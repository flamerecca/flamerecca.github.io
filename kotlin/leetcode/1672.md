## Kotlin Leetcode - 1672. Richest Customer Wealth

```kotlin
class Solution {
    fun maximumWealth(accounts: Array<IntArray>): Int {
    }
}
```

## 解題思路

這題是很單純的陣列加總

透過 `map` 可以將這段邏輯

壓縮在單行之內處理完畢

## Kotlin 參考解答

單行之內處理完畢的寫法如下

```kotlin
class Solution {
    fun maximumWealth(accounts: Array<IntArray>) = 
        accounts.map { it.sum() }.max()
}
```
