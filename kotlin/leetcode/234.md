## 解題思路

這邊我們利用 kotlin List 支援 reversed 的特性

先將輸入的 Linked List 轉換成 List

然後比較 List 的反轉是否和原本相同

## Kotlin 參考解答

使用上面的演算法得到結果如下

```kotlin
class Solution {
    fun isPalindrome(head: ListNode?): Boolean {
        val result = mutableListOf<Int>()
        var cur = head
        while (cur != null) {
            result.add(cur.`val`)
            cur = cur.next
        }
        return result == result.reversed()
    }
}
```
