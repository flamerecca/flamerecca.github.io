## 解題思路

我們觀察後發現

二的冪次的二進位剛好都是 `1(0*)` 這樣的結構

所以我們可以利用  bit operation 

快速的比對出解答

## Kotlin 參考解答

<details>
  <summary>點擊展開解答</summary>


```kotlin
class Solution {
    fun isPowerOfTwo(n: Int): Boolean {
        if(n <= 0){
            return false
        }
        return (n and (n-1)) == 0
    }
}
```

或者可以利用 `when` 縮減成單個表達式

```kotlin
class Solution {
    fun isPowerOfTwo(n: Int) = when {
        n <= 0 -> false
        else -> (n and (n-1)) == 0
	}
}
```
