## Kotlin 參考解答

參考解法如下

```kotlin
class Solution {
    fun minMovesToSeat(seats: IntArray, students: IntArray): Int {
        val seatsArray = seats.sorted()
        val studentsArray = students.sorted()
        var maxValue = 0
        for (i in seats.indices) {
            maxValue += Math.abs(seatsArray[i] - studentsArray[i])
        }
        return maxValue
    }
}
```
