## 解題思路

搭配 scope function 和 `when` 

以單一表達式解決的方式如下

```kotlin
class Solution {
    fun mergeTrees(t1: TreeNode?, t2: TreeNode?): TreeNode? {
        return when {
            t1 == null -> t2
            t2 == null -> t1
            else -> t1.apply {
                `val` += t2.`val`
                left = mergeTrees(left, t2.left)
                right = mergeTrees(right, t2.right)
            }
        }
    }
}
```
