## 解題思路

這一題考的是對樹的處理

不過他的樹只會有三個節點，所以可以在有限的狀態下寫完

## Kotlin 參考解答

單一表示式的寫法如下

```kotlin
class Solution {
    fun checkTree(root: TreeNode?): Boolean = root?.`val` == root?.left?.`val`!! + root.right?.`val`!!
}
```
