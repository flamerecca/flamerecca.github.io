class Solution {
    fun findMin(nums: IntArray): Int {

## 解題思路

這題很簡單，我們可以利用 kotlin 內建

找出陣列內最小元素的函數

這題比較麻煩的是 leetcode 目前（2022）的 kotlin 版本是 1.4

這個版本使用的函數是 `min`

如果要在比較新的版本運作

使用的函數是  `minOrNull`

## Kotlin 參考解答
leetcode 上的解答

```kotlin
class Solution {  
    fun findMin(nums: IntArray) = nums.min()  
}
```

自己電腦上撰寫的解答

```kotlin
class Solution {  
    fun findMin(nums: IntArray) = nums.minOrNull()  
}
```
