## Kotlin 參考解答

  <summary>點擊展開解答</summary>

```kotlin
class Solution {
    fun isPalindrome(x: Int): Boolean {
        if (x < 0) {
            return false
        }
        return x.toString().reversed() == x.toString()
    }
}
```

也可以用 `when` 改寫成單一表達式

```kotlin
class Solution {  
    fun isPalindrome(x: Int) = when {  
        x < 0 -> false  
        else -> x.toString().reversed() == x.toString()  
    }  
}
```
