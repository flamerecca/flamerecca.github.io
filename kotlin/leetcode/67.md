## 解題思路

比較基本的思路是實作二進位的演算法

在 Kotlin 我們可以偷懶一下

利用 `toBigInteger` 將字串轉換成數值

再使用 `toString` 將數值轉換回二進位字串

class Solution {
    fun addBinary(a: String, b: String) = 
    (a.toBigInteger(2) + b.toBigInteger(2)).toString(2)
}
