## Kotlin Leetcode - 202. Happy Number

```kotlin
class Solution {
    fun isHappy(n: Int): Boolean {
    }  
}
```

## 解題思路

這題是數字運算的題目

用 `toString()` 跟 `sumBy()` 可以很快的做出答案

## Kotlin 參考解答

```kotlin
class Solution {
    fun isHappy(n: Int): Boolean {
        var r = n
        while (r != 1 && r != 4) r = r.toString().sumBy { (it - '0') * (it - '0') }
        return r == 1
    }
}
```
