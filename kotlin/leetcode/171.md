    }

## 解題思路

這一題考的是對字串的處理

我們可以用 `fold()` 

將整個字串收攏為單一數字

## Kotlin 參考解答

```kotlin
class Solution {
    fun titleToNumber(columnTitle: String): Int = columnTitle.fold(0) { acc, ch -> 26 * acc + (ch - 'A').toInt() + 1 }
}
```
