## 解題思路

來找出石頭內是否有對應的寶石

## 解答

<details>
  <summary>點擊展開解答</summary>
```kotlin
class Solution {
    fun numJewelsInStones(jewels: String, stones: String): Int {
        var ans = 0
        for (stone in stones) {
            if (jewels.contains(stone)) {
                ans++
            }
        }
        return ans
    }
}
```


這個邏輯可以更加簡化為一行

```kotlin
class Solution {
    fun numJewelsInStones(jewels: String, stones: String) = stones.count { it in jewels }
}
```
